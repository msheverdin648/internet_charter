

Пользователь:
    id -> int
    Имя Фамилия -> str
    Почта -> str
    Этапы пройденные -> many_to_many relation
    Этапы не пройденные -> many_to_many relation
    Сертификат -> bool (Доступен/Не доступен)

Тесты
    id -> int
    Название теста -> str
    Задания -> many_to_many relation

Задание
    id -> int
    Тип вопроса -> str (будет выбираться из выпадающего списка)
    Текст вопроса -> str
    Правильный ответ -> one_to_many relation
    Возможные ответы -> many_to_many relation
    
Ответы
    id -> int
    Текст ответа -> str

Аватары для пользователей

    id -> int
    картинка -> file


Этап 
    id -> int
    Название этапа -> str
    Краткое описание -> str
    картинка/иконка -> file
    Информация? -> (пока не известно в каком виде она будет у нас храниться) str или file
    Тест -> one_to_one relation
    Доступ без авторизации -> bool (Доступен/Не доступен)
