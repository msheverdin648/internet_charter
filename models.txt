

Пользователь:
    id -> int

    ######################
    Имя -> str
    Фамилия -> str
    Отчество (если есть) -> str
    ################################## - ( Можно объеденить в одно поле ФИО)
    Возраст/дата рождения (если это необходимо) -> int or date
    Почта -> str
    Этапы пройденные -> many_to_many relation
    Этапы не пройденные -> many_to_many relation
    Сертификат -> bool (Доступен/Не доступен)



Тесты
    id -> int
    Название теста -> str
    Вопрос/ответ -> many_to_many relation

Вопросы
    id -> int
    Тип ответа -> str (будет выбираться из выпадающего списка)
    Вопрос -> str
    Правильный ответ(можно ввести несколько, на случай чекбоксов) -> str
    


Аватары для пользователей

    id -> int
    картинка -> file


Этап 
    id -> int
    Название этапа -> str
    Краткое описание -> str
    картинка/иконка -> file
    Информация? -> (пока не известно в каком виде она будет у нас храниться) str или file
    Тест -> one_to_one relation
    Доступ без авторизации -> bool (Доступен/Не доступен)
